# CleanCo – Laundry Service Web App

A simple web application for a laundry service that lets users browse services, add them to a cart, and book an appointment.
Features

# Responsive navigation bar

Hero section with a "Book a Service Today" button
Service stats (15+ services, 240+ customers, 2+ years)
Booking section with cart and order form
Email confirmation via EmailJS
Quality highlights section
Newsletter subscription
Footer with contact details and social links

# How to Run

Just open index.html in any browser. No installation needed.
Setting Up Email

# Sign up at emailjs.com

Create a service and an email template
In index.html, replace the following placeholders:

YOUR_EMAILJS_PUBLIC_KEY
YOUR_SERVICE_ID
YOUR_TEMPLATE_ID

# Tech Used

HTML, CSS, JavaScript
EmailJS (email confirmation)
